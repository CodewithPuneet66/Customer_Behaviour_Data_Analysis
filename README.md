# 🛍️ Customer Shopping Behavior Analysis | End-to-End Data Analytics Project

> An end-to-end data analytics project demonstrating **Python, SQL, and Power BI** to transform raw retail data into actionable business insights through data cleaning, analysis, and interactive dashboards.

---

## 📌 Project Summary

Businesses generate large volumes of customer transaction data, but extracting meaningful insights requires effective data analysis.

In this project, I analyzed a customer shopping behavior dataset to identify purchasing trends, customer demographics, spending patterns, and seasonal sales insights. The project follows a complete analytics workflow—from data cleaning and exploratory analysis to SQL-based business queries and an interactive Power BI dashboard.

This project demonstrates the practical skills expected from a **Data Analyst**, including data preprocessing, business analysis, dashboard development, and insight generation.

---

## 🎯 Business Problem

Retail companies need to understand:

* Which customer segments spend the most?
* Which product categories generate the highest revenue?
* How do purchasing habits vary by age and gender?
* Which payment methods are most preferred?
* How does seasonality affect sales?
* What factors influence customer satisfaction?

This project answers these questions using data analytics techniques.
